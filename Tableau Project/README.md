# Tableau Data Visualization Project

## Overview
Interactive Tableau dashboard created to showcase data visualization and storytelling skills.

## Data
This project uses publicly available flight delay data, including:
- Airlines reference data
- Airport reference data
- Flight-level delay records

The data is provided in Excel format and joined within Tableau for analysis.

## Business Questions
- Which airlines experience the highest average departure delays?
- How do departure delays relate to arrival delays?
- What overall patterns exist in U.S. flight delays?

## Dashboard Features
- Interactive filters for airline and delay ranges
- Multiple visualization types to highlight trends and relationships
- Dashboard and worksheet-level views to support exploratory analysis
